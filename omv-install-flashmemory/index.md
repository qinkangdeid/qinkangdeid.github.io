# OMV安装 flashmemory

我的OMV是装在了U盘上的，该插件可以将系统运行产生的临时文件存进内存，降低磁盘的读写次数，从而达到延长系统盘的使用寿命。

>  flashmemory属于omv拓展(omv-extras)插件，如果没有安装拓展包是搜索不到的
>  如果你还没有安装omv-extras的话，请先安装[OMV安装omv-extras](https://www.jianshu.com/p/4a0aa7e48515)
